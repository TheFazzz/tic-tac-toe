# X's and O's Showdown

## Description

Web tic tac toe game written in React

## Getting Started

### Dependencies

* Windows OS
* Node JS

### Installing

* Setup Node modules: (run command)
```
npm install
```

### Executing program

* How to run: (run command)
```
npm start
```
* ctrl-c to stop server
